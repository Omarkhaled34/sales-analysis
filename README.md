# sales-analysis
Data warehouse analytics queries for sales performance, customer segmentation, and product analysis using SQL


### What's included
- Monthly sales performance summary (revenue, customers, volume)
- Cumulative sales trends over time using window functions
- Yearly product performance vs. average and previous year (YoY analysis)
- Category contribution to overall sales (percentage breakdown)
- Product segmentation by cost range
- Customer segmentation: VIP / Regular / New based on spending and lifespan

### Tech
SQL (T-SQL) · Window Functions · CTEs · Star Schema (fact_sales, dim_products, dim_customers)
